# Cantilever
Расчет эпюр балки 

Удалите доп скобки  '<!- -->' в  файле index.html из двух строк ниже на выбор  для того что бы программа работала. 
Для первой строчки нужно распоковать папку konva
<br>
для второй строчки нужно иметь подключение к интернету
<br>
<script src="/konva/konva.js"></script>
<br>
<script src="https://cdn.rawgit.com/konvajs/konva/1.7.6/konva.js"></script>
